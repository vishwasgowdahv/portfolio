## 🖥 About This Portfolio Site

This repository contains the source code for my interactive developer portfolio. It is designed to act as an immersive, highly performant experience showcasing my work and background.

### ⚡ Key Features of this Site
* **Modern CSS & Styles**: Styling implemented with vanilla modern CSS and Tailwind CSS v4.
* **Smooth Animations**: Animated transitions and layouts using `framer-motion`.
* **Smooth Scrolling**: Momentum-based scroll powered by `@studio-freight/lenis`.
* **Dark Mode**: Fully implemented visual theme selector context.
* **Data-Driven Layouts**: All content is dynamically populated from a single configuration file at `src/data/portfolioData.js`.

### ⚙️ Local Development Setup

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vishwasgowdahv/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`.

4. **Build for production:**
   ```bash
   npm run build
   ```
   This creates a static production build in the `dist/` directory.
